> A gem to get a cute ASCII cuchi easily aka ASCUCHIII.


## Install

```sh
gem install ascuchiii
```

## Usage

```sh
# Default to colorful and width 50
ascuchiii

# If you dont want colors
ascuchiii -c false 

# If you dont different widths 
ascuchiii -w 100 
```
